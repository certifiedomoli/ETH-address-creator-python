# ETH-address-creator-python
 create Ethereum address from 12phrase pass with python
welcome!
This is a digital currency wallet recovery project with Python! Using the original version of this program that will be released soon, you can recover all types of wallets and get their passwords under certain conditions.
_______________________________________________
Dependencies to be installed:
-PyQt5
-mnemonic
-eth_account
-time
-sys
-logging
_______________________________________________
This is a basic version of the program that simply takes a 12-word password from the user and displays the Ethereum address of that password.
_______________________________________________
The next update includes:   (coming soon)
-Checking the current wallet
-Checking wallet transactions
-Detecting the closest words to the user's current password to guess different phrases
-Connecting to the database
-Generating all different versions of a 12-word phrase
-Ability to restore the wallet if the order of the passwords is lost
-Ability to restore the wallet if you are suspicious of a few words
-Ability to crack a lucky wallet
